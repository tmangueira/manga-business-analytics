Identificar e definir o problema são as principais etapas para se encontrar uma solução.

# Como definir um problema?
1) Explore o cenário atual
- Quais são os impactos e consequências em não resolver o problema?
- Quais as possíveis causas?
- Qual a natureza do problema?

2) Explique o problema
- Quando você identificar e definir o problema, explique o problema para que você possa compreender melhor sobre ele.

3) Questione a si mesmo
- Este é realmente o problema?
- Refaça cenários para validar se não se trata de uma consequência do problema.
